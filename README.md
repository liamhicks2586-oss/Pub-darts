# 🎯 Pub Darts Scoreboard

A professional live pub darts scoring system built as an iPhone web app. Perfect for pubs, bars, and darts clubs!

## ✨ Features

✅ **iPhone Web App** - Install directly to home screen, no App Store needed  
✅ **Live Scoring** - Score 501, 601, 701, 801, 901, or 1001  
✅ **Flexible Scoring Modes** - Score individual darts OR enter total for 3 darts  
✅ **Custom Score Input** - Enter any score 0-180 manually  
✅ **Dual Display** - Show scores on pub TV with Sky Sports-style graphics  
✅ **Real-time Sync** - Scores update instantly across all devices  
✅ **Offline Support** - App works offline once installed  
✅ **No PC Required** - Everything runs from your iPhone  

## 🎮 Scoring Modes

Each player can choose their preferred scoring method:

### Single Dart Mode
- Record each dart individually as you throw
- Automatically switches to next player after 3 darts
- Perfect for traditional scoring
- Dart counter shows progress: 0/3, 1/3, 2/3

### Total 3 Darts Mode
- Enter the combined total of 3 darts as ONE number
- Auto-switches after entering total
- Faster for quick input
- Dart counter shows: 1/1 (one entry)

### Custom Score Entry
- Type any score (0-180) directly into the custom input field
- Tap "Add" to record
- Works in either mode

## 🚀 Quick Start

### Installation on iPhone

1. **Open Safari** on your iPhone
2. Go to: `https://liamhicks2586-oss.github.io/pub-darts/`
3. Tap **Share** (⬆️ icon at bottom)
4. Select **"Add to Home Screen"**
5. Name it: **🎯 Pub Darts**
6. Tap **"Add"**

### Using the App

**Scoring Device (Your iPhone):**
1. Open the 🎯 Pub Darts app
2. Enter player names
3. Select starting score (501-1001)
4. Tap "START GAME"
5. Choose scoring mode (Single Dart or Total 3)
6. Record darts using preset buttons or custom input
7. Scores auto-switch between players

**Display Screen (Pub TV):**
1. Open any browser on the pub TV/monitor
2. Go to: `https://liamhicks2586-oss.github.io/pub-darts/display`
3. Scores update in REAL-TIME like Sky Sports broadcasts

## 📋 Game Rules

- **Scoring**: Players take turns recording dart scores
- **Single Dart**: Score each dart individually (0-60 typically)
- **Total 3 Darts**: Enter combined total of 3 throws (0-180 max)
- **Custom Score**: Enter any score manually via input field
- **Bust**: If score goes below 0, it resets to 0 for that turn
- **Winner**: First player to reach exactly 0 wins
- **Undo**: Tap "UNDO" to undo last entry
- **Reset**: Tap "RESET" to clear current turn
